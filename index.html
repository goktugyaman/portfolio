Storage Location Optimizer

Real project built at STIGA S.p.a — for each existing storage location (plant), the model fetches driving distances to all assigned ship-to customers via Google Distance Matrix API, weights them by order volume (units shipped), and computes the weighted centroid as the optimal warehouse location. The gap between current and optimal location quantifies logistics inefficiency.
